# LM2596 Efficiency Calculator

TypeScript-based calculator for analyzing LM2596 buck converter efficiency. Includes a pure TypeScript core module, browser API, and interactive D3/React visualizations.

## Quick Start

**Install dependencies:**
```bash
npm install
```

**Run apps:**
```bash
npm run dev:d3      # D3 visualization app
npm run dev:react   # React app
npm test            # Run tests
```

**Build:**
```bash
npm run build       # Build all targets
```

See [browser-demo.html](browser-demo.html) for browser usage examples.
